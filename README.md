A desktop-based digital notepad application developed using Java Swing that allows users to create, edit, and manage text files with a simple and user-friendly interface.

---

## Features
- Create, open, and save text files
- Cut, copy, paste, and select all options
- Real-time status bar to show cursor position
- Dark Mode toggle for better user experience
- Simple and intuitive UI
